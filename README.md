# JioTV plugin
This plugin supports jiotv playback in kodi with catchup. Sony channels due to recent changes requires premium specific plans, so if anyone have pubads or dai link for sony channels kindly open a issue with feature request tag and i would add them in the plugin.

# Few Common Erros With Solutions
## "Key Error: Result" When playing anything 
Jiotv limits number of specific devices to each account your device limit might be reached , logout from here login in jiotv app then logout from there and login here fixes see issue https://github.com/AmanCode22/plugin.video.aman.jiotv/issues/3 for more.
If still not fixed then open issue.
